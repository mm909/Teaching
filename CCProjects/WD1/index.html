Welcome to your first web development project! In this project we will go through many aspects of basic web development and how they work together to make a complete and pretty looking website with many interactable features.

Getting Started

You may have done some HTML and CSS in CodeCombat, but this is to show you what it looks like when it’s not in a game.

The first important thing to keep in mind, is that you have to have a project folder for your site

Find where you would like to place your project folder.

Right Click to open the context menu -> Hover over “New” -> Press on “Folder”.


Rename the folder to “firstwebsite”

This will be where you place all of your files for your website.

Files

In any coding project, there is always that starting point where your project will go to first to run code. In websites, that’s called “index”. The index file is the first page that should open if someone were to go to your site
